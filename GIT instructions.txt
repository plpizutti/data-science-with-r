GIT Instructions for the project:

- repository name: data-science-with-r
- the head branch is main
- to create new branch:
  git checkout -b "branch-name"

- to push new branches to github repo:
  git push --set-upstream origin "branch-name"

- to push new files / changes:
  Git Bash:
  git add "file-name.ext"
  git commit -m "message"
  git push

  Github:
  Merge changes into main

- After uploading the changes to the main branch:
  delete branch
